What you will see

  1. Start where you left off. Open the project instead of starting from a blank chat.
  2. Keep the important things together. Files and decisions stay with the project.
  3. Ask a question and keep the context. Get help without losing the rest of the work.
  4. See what is ready. Notice what is finished, what is being checked, and what is only a test.
  5. Use the same project on a small screen. The work stays readable on mobile.

What this video does not show

This is a local preview, not a download or product release. The proof and roadmap page explains what still needs to happen.
